Последний официальный плагин для bloc и flutter_bloc, который мы не разобрали.

bloc_test - это пакет для тестирования блоков и кубитов. Он содержит готовые методы для написания тестов для бизнес-компонентов.

В нашем проекте для тестирования bloc_example теперь есть тесты, они уже находятся в мастере.

Написал тесты для следующих кейсов:

— Инициализация, постраничная загрузка и загрузка последней страницы с данными для экрана с пользователями. — Инициализация, загрузки, logIn и logOut профиля.

Помимо создания тестов для блоков, было сделано:

— Конфиг для тестов в GitHub Actions. Сам файл тут, внизу будет секция tests. — Моки для hydrated_bloc и репозиториев из проекта. Реализованы через mockito. — Обновление Флаттера и рефакторинг под новые синтаксические фичи Дарта.

Ознакомиться со всеми изменениями можно по [ссылке](https://github.com/fluttermiddlepodcast/bloc_example/tree/master/test).
